HTML helpers (Plug-ins)

Helper applications (plug-ins) are computer programs that extend the standard functionality of a web browser. Examples of well-known plug-ins are Java applets

Plugins can be add to web pages with the object tag or the embed tag.


Object element is supported by all browsers, it defines an embeded object within an HTML document - it is used to embed plugin-s like Java applets, pdf Readers, flash players.
