What is HTML
	- standard markup language for creating Web pages.
	- Hyper Text Markup Language
	- html elements are represented by tags - that are rendered by browser 
	
<!Doctype> 
	- declaration of <!DOCTYPE> represents the document type, and help 
		browsers to display web pages correctly.
	- it's not case sensitive
	- declaration for HTML5 is <!DOCTYPE html>
	
	--- Tips 
		all html tags are not case sensitive - <p> == <P>
		html5 standard doesn't require quotes over attribute values - but it demands quotes for stricter document types like xhtml
